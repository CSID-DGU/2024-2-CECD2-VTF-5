## 📝 프로젝트 설명
AI 구술 자서전 서비스는 고령자를 대상으로 음성 인식 기술을 활용해 편리하게 자서전을 작성할 수 있는 서비스입니다. 이 서비스는 음성을 텍스트로 변환(STT)하고, 대화형 AI를 통해 질문에 답하며 자서전 내용을 풍부하게 만들 수 있습니다.
